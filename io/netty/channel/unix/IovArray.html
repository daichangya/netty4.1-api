io.netty.channel.unix

Class IovArray

  • java.lang.Object
    • io.netty.channel.unix.IovArray
  • All Implemented Interfaces:
    ChannelOutboundBuffer.MessageProcessor 

    public final class IovArray
extends java.lang.Object
implements ChannelOutboundBuffer.MessageProcessor
    表示一个结构数组的数组，因此可以通过JNI直接传递，而不需要再做任何数组拷贝。 缓冲区直接写入直接内存以匹配struct iov。 也参见man writev 。 
      struct iovec {
   void  *iov_base;
   size_t iov_len;
 };
    另见Efficient JNI programming IV: Wrapping native data objects 。

      • maxBytes

        public void maxBytes(long maxBytes)
        设置可通过add(ByteBuf)添加到此IovArray的最大字节数 。

        这不会影响IovArray的现有状态，仅适用于后续调用add(ByteBuf) 。

        为了确保取得一些进展，即使尺寸超过此值，至少也会接受一个ByteBuf 。

        参数
        maxBytes - 可通过add(ByteBuf)添加到此IovArray的最大字节数 。
