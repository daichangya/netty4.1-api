Package io.netty.channel.socket

抽象扩展核心通道API的TCP和UDP套接字接口。

See: 描述

  • 接口摘要  
    接口 描述
    DatagramChannel
    一个UDP / IP Channel 。
    DatagramChannelConfig
    A ChannelConfig为DatagramChannel 。
    DuplexChannel
    双面Channel ，可双面关闭。
    ServerSocketChannel
    接受传入TCP / IP连接的TCP / IP ServerChannel 。
    ServerSocketChannelConfig
    A ChannelConfig为ServerSocketChannel 。
    SocketChannel
    一个TCP / IP套接字Channel 。
    SocketChannelConfig
    A ChannelConfig为SocketChannel 。
  • 类摘要  
    Class 描述
    ChannelInputShutdownEvent
    这将被解雇，并传递到特殊事件ChannelInboundHandler.userEventTriggered(ChannelHandlerContext, Object)种方法一旦输入SocketChannel停工，并在SocketChannelConfig.isAllowHalfClosure()方法返回true 。
    ChannelInputShutdownReadComplete
    表示频道输入端的用户事件已关闭，我们试图再次关闭它。
    ChannelOutputShutdownEvent
    这将被解雇，并传递到特殊事件ChannelInboundHandler.userEventTriggered(ChannelHandlerContext, Object)种方法一次的输出SocketChannel被关闭。
    DatagramPacket
    用于DatagramChannel与远程对等方进行通信的消息容器。
    DefaultDatagramChannelConfig
    默认的DatagramChannelConfig实现。
    DefaultServerSocketChannelConfig
    默认的ServerSocketChannelConfig实现。
    DefaultSocketChannelConfig
    默认的SocketChannelConfig实现。
  • 枚举摘要  
    Enum 描述
    InternetProtocolFamily
    Internet协议（IP）系列使用字节DatagramChannel
  • 异常摘要  
    异常 描述
    ChannelOutputShutdownException
    用于在通道的输出已关闭时发生挂起写入失败。
