io.netty.channel.pool

Interface ChannelPool

  • All Superinterfaces:
    java.lang.AutoCloseable，java.io.Closeable
    所有已知实现类：
    FixedChannelPool ， SimpleChannelPool 

    public interface ChannelPool
extends java.io.Closeable
    允许获取并释放Channel，并充当这些池的作用。

    • 方法详细信息

      • acquire

        Future<Channel> acquire()
        获得Channel从这个ChannelPool 。 一旦获得成功，通知返回的Future ，否则失败。 重要的是，即使Channel明确关闭，获得的数据也会再次发布到池中。

      • acquire

        Future<Channel> acquire(Promise<Channel> promise)
        获得Channel从这个ChannelPool 。 一旦获得成功，通知给定的Promise ，否则失败。 重要的是，即使Channel明确关闭，获得的数据也会再次发布到池中。

      • release

        Future<java.lang.Void> release(Channel channel)
        释放Channel回到此ChannelPool 。 一旦发布成功，通知返回的Future ，否则失败。 失败时， Channel将自动关闭。

      • release

        Future<java.lang.Void> release(Channel channel,
                               Promise<java.lang.Void> promise)
        释放一个Channel回到这个ChannelPool 。 一旦发布成功，通知给定的Promise ，否则失败。 失败时， Channel将自动关闭。
