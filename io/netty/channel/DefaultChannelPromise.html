Class DefaultChannelPromise

  • java.lang.Object

    public class DefaultChannelPromise
extends DefaultPromise<java.lang.Void>
implements ChannelPromise
    默认的ChannelPromise实现。 建议使用ChannelOutboundInvoker.newPromise()创建新的ChannelPromise，而不是显式调用构造函数。
