Package io.netty.handler.ipfilter

打包来过滤IP地址（允许/拒绝）。

See: 描述

  • 接口摘要  
    接口 描述
    IpFilterRule
    实现这个接口来创建新的规则。
  • 类摘要  
    Class 描述
    AbstractRemoteAddressFilter<T extends java.net.SocketAddress>
    该类提供了根据IP地址接受或拒绝新的Channel的功能。
    IpSubnetFilterRule
    使用此类为RuleBasedIpFilter创建规则，将IP地址分组为子网。
    RuleBasedIpFilter
    该类允许根据传递给其构造函数的IpFilterRule过滤新的Channel 。
    UniqueIpFilter
    这个类可以确保在任何时候每个IP地址最多只有一个Channel连接到服务器。
  • 枚举摘要  
    Enum 描述
    IpFilterRuleType
    在IpFilterRule中用于决定是否允许或拒绝匹配的IP地址进行连接。
