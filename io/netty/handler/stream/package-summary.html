Package io.netty.handler.stream

写入非常大的数据流，不会花费大量内存，也不会获得 OutOfMemoryError 。

See: 描述

  • 接口摘要  
    接口 描述
    ChunkedInput<B>
    ChunkedWriteHandler消耗的长度不定的数据流。
  • 类摘要  
    Class 描述
    ChunkedFile
    一个ChunkedInput通过块从文件块中获取数据。
    ChunkedNioFile
    A ChunkedInput ，使用NIO FileChannel通过块从文件块中提取数据。
    ChunkedNioStream
    一个ChunkedInput按块ReadableByteChannel块中的数据。
    ChunkedStream
    一个ChunkedInput ，按块InputStream块中的数据。
    ChunkedWriteHandler
    ChannelHandler增加了对异步写入大数据流的支持，既不花费大量内存也不获取OutOfMemoryError 。

Package io.netty.handler.stream Description

写入非常大的数据流，不会花费大量内存，也不会获得OutOfMemoryError 。 有关详细示例，请参阅io.netty.example.http.file 。
