io.netty.handler.stream

Class ChunkedNioFile

  • java.lang.Object
    • io.netty.handler.stream.ChunkedNioFile
  • All Implemented Interfaces:
    ChunkedInput < ByteBuf > 

    public class ChunkedNioFile
extends java.lang.Object
implements ChunkedInput<ByteBuf>
    A ChunkedInput ，使用NIO FileChannel通过块从文件块中提取数据。

    如果您的操作系统支持zero-copy file transfer，例如sendfile() ，则可能需要使用FileRegion 。
