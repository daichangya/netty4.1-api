io.netty.handler.stream

Class ChunkedStream

  • java.lang.Object
    • io.netty.handler.stream.ChunkedStream
  • All Implemented Interfaces:
    ChunkedInput < ByteBuf > 

    public class ChunkedStream
extends java.lang.Object
implements ChunkedInput<ByteBuf>
    A ChunkedInput ，按大块提取InputStream块中的数据。

    请注意，将数据提供给ChunkedStream的InputStream实例必须尽可能准确地实现InputStream.available() ，而不是使用默认实现。 否则， ChunkedStream会生成许多太小的块或经常不必要的块。
