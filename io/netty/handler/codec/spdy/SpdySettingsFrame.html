    • 方法详细信息

      • ids

        java.util.Set<java.lang.Integer> ids()
        返回设置ID的Set 。 该集合的迭代器将按照升序返回ID。

      • isSet

        boolean isSet(int id)
        如果设置ID有值，则返回 true 。

      • getValue

        int getValue(int id)
        返回设置ID的值。 如果设置ID未设置，则返回-1。

      • setValue

        SpdySettingsFrame setValue(int id,
                           int value)
        设置设置ID的值。 ID不能为负数，不能超过16777215。

      • setValue

        SpdySettingsFrame setValue(int id,
                           int value,
                           boolean persistVal,
                           boolean persisted)
        设置设置ID的值。 设置是否应该保持设置（应该仅由服务器设置）。 设置是否保持设置（应该仅由客户端设置）。 ID不能为负数，不能超过16777215。

      • removeValue

        SpdySettingsFrame removeValue(int id)
        删除设置ID的值。 删除设置的所有持久性信息。

      • isPersistValue

        boolean isPersistValue(int id)
        如果此设置应该true则返回true 。 如果此设置不应被false或者设置ID没有值，则返回false 。

      • setPersistValue

        SpdySettingsFrame setPersistValue(int id,
                                  boolean persistValue)
        设置是否应该保留此设置。 如果设置ID没有值，则不起作用。

      • isPersisted

        boolean isPersisted(int id)
        如果此设置持续，则返回true 。 如果此设置不应被false或者设置ID没有值，则返回false 。

      • setPersisted

        SpdySettingsFrame setPersisted(int id,
                               boolean persisted)
        设置此设置是否持续。 如果设置ID没有值，则不起作用。

      • clearPreviouslyPersistedSettings

        boolean clearPreviouslyPersistedSettings()
        如果以前 true设置应该清除，则返回 true 。

      • setClearPreviouslyPersistedSettings

        SpdySettingsFrame setClearPreviouslyPersistedSettings(boolean clear)
        设置是否应清除以前保留的设置。
