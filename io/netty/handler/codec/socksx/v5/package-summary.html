Package io.netty.handler.codec.socksx.v5

编码器，解码器及其相关的SOCKSv5协议消息类型。

See: 描述

  • 接口摘要  
    接口 描述
    Socks5AddressDecoder
    将SOCKS5地址字段解码为其字符串表示形式。
    Socks5AddressEncoder
    将SOCKS5地址编码为二进制表示。
    Socks5CommandRequest
    SOCKS5请求详细消息，如 the section 4, RFC1928中所定义。
    Socks5CommandResponse
    如 the section 6, RFC1928中所定义的对SOCKS5请求详细消息的 响应 。
    Socks5InitialRequest
    如 the section 3, RFC1928中定义的初始SOCKS5认证方法选择请求。
    Socks5InitialResponse
    初始SOCKS5认证方法选择请求，如 the section 3, RFC1928中所定义。
    Socks5Message
    所有SOCKS5协议消息实现的标记接口。
    Socks5PasswordAuthRequest
    如 the section 2, RFC1929中所定义的用于用户名 - 密码认证的SOCKS5子 协商请求 。
    Socks5PasswordAuthResponse
    如 the section 2, RFC1929中所定义的用于用户名 - 密码认证的SOCKS5子 协商响应 。
  • 类摘要  
    Class 描述
    AbstractSocks5Message
    摘要Socks5Message 。
    DefaultSocks5CommandRequest
    默认Socks5CommandRequest 。
    DefaultSocks5CommandResponse
    默认Socks5CommandResponse 。
    DefaultSocks5InitialRequest
    默认Socks5InitialRequest 。
    DefaultSocks5InitialResponse
    默认Socks5InitialResponse 。
    DefaultSocks5PasswordAuthRequest
    默认Socks5PasswordAuthRequest 。
    DefaultSocks5PasswordAuthResponse
    默认Socks5PasswordAuthResponse 。
    Socks5AddressType
    地址类型Socks5CommandRequest和Socks5CommandResponse 。
    Socks5AuthMethod
    SOCKS5的认证方法。
    Socks5ClientEncoder
    将客户端Socks5Message编码为ByteBuf 。
    Socks5CommandRequestDecoder
    从入境ByteBuf解码单个Socks5CommandRequest 。
    Socks5CommandResponseDecoder
    从入站ByteBuf解码单个Socks5CommandResponse 。
    Socks5CommandStatus
    状态Socks5CommandResponse 。
    Socks5CommandType
    类型Socks5CommandRequest 。
    Socks5InitialRequestDecoder
    从入站ByteBuf解码单个Socks5InitialRequest 。
    Socks5InitialResponseDecoder
    从入站ByteBuf解码单个Socks5InitialResponse 。
    Socks5PasswordAuthRequestDecoder
    从入站ByteBuf解码单个Socks5PasswordAuthRequest 。
    Socks5PasswordAuthResponseDecoder
    从入站ByteBuf解码单个Socks5PasswordAuthResponse 。
    Socks5PasswordAuthStatus
    Socks5PasswordAuthResponse的状态。
    Socks5ServerEncoder
    将服务器端Socks5Message编码为ByteBuf 。

Package io.netty.handler.codec.socksx.v5 Description

Encoder, decoder and their related message types for SOCKSv5 protocol.
