Package io.netty.handler.codec.memcache.binary

Memcache二进制协议的实现和接口。

See: 描述

  • 接口摘要  
    接口 描述
    BinaryMemcacheMessage
    定义二进制Memcache消息的接口，为BinaryMemcacheRequest和BinaryMemcacheResponse提供公共属性。
    BinaryMemcacheRequest
    代表完整的BinaryMemcacheRequest ，其中包含标题和可选的关键字和附加内容。
    BinaryMemcacheResponse
    代表完整的BinaryMemcacheResponse ，其中包含标题和可选的关键字和附加内容。
    FullBinaryMemcacheRequest
    A BinaryMemcacheRequest也包含内容。
    FullBinaryMemcacheResponse
    A BinaryMemcacheResponse也包含内容。
  • 类摘要  
    Class 描述
    AbstractBinaryMemcacheDecoder<M extends BinaryMemcacheMessage>
    BinaryMemcacheRequest和BinaryMemcacheResponse的解码器。
    AbstractBinaryMemcacheEncoder<M extends BinaryMemcacheMessage>
    将二进制内存缓存消息编码为字节的MessageToByteEncoder 。
    AbstractBinaryMemcacheMessage
    BinaryMemcacheMessage的默认实现。
    BinaryMemcacheClientCodec
    结合了正确编码器和解码器的客户端编解码器。
    BinaryMemcacheObjectAggregator
    memcache二进制协议的对象聚合器。
    BinaryMemcacheOpcodes
    代表BinaryMemcacheMessage中可能出现的所有操作码。
    BinaryMemcacheRequestDecoder
    负责解码请求特定报头的解码器部分。
    BinaryMemcacheRequestEncoder
    负责编码请求标头的编码器部分。
    BinaryMemcacheResponseDecoder
    负责解码响应头的解码器。
    BinaryMemcacheResponseEncoder
    负责编码响应头的编码器。
    BinaryMemcacheResponseStatus
    包含一个BinaryMemcacheResponse可以返回的所有可能的状态值。
    BinaryMemcacheServerCodec
    完整的服务器编解码器，结合了正确的编码器和解码器。
    DefaultBinaryMemcacheRequest
    BinaryMemcacheRequest的默认实现。
    DefaultBinaryMemcacheResponse
    BinaryMemcacheResponse的默认实现。
    DefaultFullBinaryMemcacheRequest
    FullBinaryMemcacheRequest的默认实现。
    DefaultFullBinaryMemcacheResponse
    FullBinaryMemcacheResponse的默认实现。
