Package io.netty.handler.codec.memcache

ascii和二进制类的通用超集。

See: 描述

  • 接口摘要  
    接口 描述
    FullMemcacheMessage
    将MemcacheMessage和LastMemcacheContent组合成一条消息。
    LastMemcacheContent
    MemcacheContent表示内容批处理结束。
    MemcacheContent
    Memcache内容块。
    MemcacheMessage
    用于ASCII和二进制消息的标记接口。
    MemcacheObject
    定义所有MemcacheObject实现的通用接口。
  • 类摘要  
    Class 描述
    AbstractMemcacheObject
    默认的MemcacheObject实现。
    AbstractMemcacheObjectAggregator<H extends MemcacheMessage>
    A ChannelHandler ，其将MemcacheMessage及其后续MemcacheContent汇总为单个MemcacheMessage而没有后续MemcacheContent 。
    AbstractMemcacheObjectDecoder
    ascii和二进制解码器的抽象超类。
    AbstractMemcacheObjectEncoder<M extends MemcacheMessage>
    编码MemcacheMessage的通用AbstractMemcacheObjectEncoder 。
    DefaultLastMemcacheContent
    LastMemcacheContent的默认实现。
    DefaultMemcacheContent
    默认的MemcacheContent实现。
