Package io.netty.handler.codec.redis

编码器，Redis解码器。

See: 描述

  • 接口摘要  
    接口 描述
    BulkStringRedisContent
    大量用于Redis分块传输编码的批量字符串。
    LastBulkStringRedisContent
    Bulk Strings的最后一块。
    RedisMessage
    RedisMessage是codec-redis的基础接口。
    RedisMessagePool
    用于缓存RedisMessage的策略接口。
  • 类摘要  
    Class 描述
    AbstractStringRedisMessage
    简单字符串或错误的抽象类。
    ArrayHeaderRedisMessage
    Redis阵列消息头。
    ArrayRedisMessage
    数组 RESP 。
    BulkStringHeaderRedisMessage
    Bulk Strings的标题为 RESP 。
    DefaultBulkStringRedisContent
    默认实现BulkStringRedisContent 。
    DefaultLastBulkStringRedisContent
    LastBulkStringRedisContent的默认实现。
    ErrorRedisMessage
    RESP的错误。
    FixedRedisMessagePool
    默认的固定redis消息池。
    FullBulkStringRedisMessage
    RESP的汇总批量字符串。
    IntegerRedisMessage
    整数 RESP 。
    RedisArrayAggregator
    将RedisMessage部分聚合为ArrayRedisMessage 。
    RedisBulkStringAggregator
    A ChannelHandler将BulkStringHeaderRedisMessage及其后续BulkStringRedisContent合并为一个FullBulkStringRedisMessage ，但没有BulkStringRedisContent 。
    RedisDecoder
    按照RESP (REdis Serialization Protocol)将Redis协议解码为RedisMessage个对象。
    RedisEncoder
    编码RedisMessage以下为字节RESP (REdis Serialization Protocol) 。
    SimpleStringRedisMessage
    简单字符串 RESP 。
  • 枚举摘要  
    Enum 描述
    RedisMessageType
    类型 RESP (REdis Serialization Protocol) 。
  • 异常摘要  
    异常 描述
    RedisCodecException
    一个异常这是通过抛出RedisEncoder或RedisDecoder 。
