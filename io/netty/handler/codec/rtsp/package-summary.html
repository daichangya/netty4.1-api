Package io.netty.handler.codec.rtsp

基于HTTP编解码器的 RTSP扩展。

See: 描述

  • 类摘要  
    Class 描述
    RtspDecoder
    解码ByteBuf s转换中表示RTSP消息HttpMessage秒。
    RtspEncoder
    将HttpMessage中表示的RTSP消息或HttpContent中的RTSP消息编码为ByteBuf 。
    RtspHeaderNames
    标准RTSP头名称。
    RtspHeaders Deprecated
    改为使用RtspHeaderNames或RtspHeaderValues 。
    RtspHeaders.Names Deprecated
    改为使用RtspHeaderNames 。
    RtspHeaders.Values Deprecated
    改为使用RtspHeaderValues 。
    RtspHeaderValues
    标准RTSP头名称。
    RtspMethods
    RTSP的请求getMethod。
    RtspObjectDecoder Deprecated
    改为使用RtspDecoder 。
    RtspObjectEncoder<H extends HttpMessage> Deprecated
    改为使用RtspEncoder 。
    RtspRequestDecoder Deprecated
    直接使用RtspDecoder
    RtspRequestEncoder Deprecated
    直接使用RtspEncoder
    RtspResponseDecoder Deprecated
    直接使用RtspDecoder
    RtspResponseEncoder Deprecated
    直接使用RtspEncoder
    RtspResponseStatuses
    getStatus代码及其对RTSP响应的描述。
    RtspVersions
    RTSP的版本。
