    public abstract class HttpObjectEncoder<H extends HttpMessage>
extends MessageToMessageEncoder<java.lang.Object>
    将HttpMessage或HttpContent编码为ByteBuf 。

    可扩展性

    请注意，此编码器设计用于扩展以实现从HTTP派生的协议，例如RTSP和ICAP 。 为了实现这种衍生协议的编码器，扩展这个类并适当地实现所有的抽象方法。
