Class HttpClientUpgradeHandler

  • All Implemented Interfaces:
    ChannelHandler ， ChannelInboundHandler ， ChannelOutboundHandler 

    public class HttpClientUpgradeHandler
extends HttpObjectAggregator
implements ChannelOutboundHandler
    用于处理与另一个协议的HTTP升级握手的客户端处理程序。 当发送第一个HTTP请求时，该处理程序将添加所有适当的标头以执行到新协议的升级。 如果升级失败（即响应不是101切换协议），则此处理程序只是将自己从管道中移除。 如果升级成功，则将管道升级到新协议。
