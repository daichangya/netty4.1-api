      • HttpVersion

        public HttpVersion(java.lang.String text,
                   boolean keepAliveDefault)
        用指定的版本字符串创建一个新的HTTP版本。 除非您正在实施派生自HTTP的协议，否则不需要创建新实例，例如RTSP和ICAP 。
        参数
        keepAliveDefault - true当且仅当连接保持活动状态，除非明确将 "Connection"标头设置为 "close" 。

      • valueOf

        public static HttpVersion valueOf(java.lang.String text)
        返回与指定协议版本字符串匹配的现有或新的HttpVersion实例。 如果指定的text等于"HTTP/1.0" ，则返回HTTP_1_0 。 如果指定的text等于"HTTP/1.1" ，则返回HTTP_1_1 。 否则，将返回一个新的HttpVersion实例。
