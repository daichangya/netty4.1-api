      • containsValue

        public boolean containsValue(java.lang.CharSequence name,
                             java.lang.CharSequence value,
                             boolean ignoreCase)
        从类复制的描述： HttpHeaders
        返回true如果与头name和value存在， false其他。 这也处理了用,分隔的多个值。

        如果ignoreCase是true则对值进行不区分大小写的比较。

        重写：
        containsValue在课程 HttpHeaders
        参数
        name - 要查找的标题的名称
        value - 标题的值查找
        ignoreCase - true然后运行一个不区分大小写的比较来比较值。 否则运行区分大小写比较来比较值。
