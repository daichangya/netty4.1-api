io.netty.handler.codec.http.cookie

Class ServerCookieDecoder

  • java.lang.Object
    • io.netty.handler.codec.http.cookie.CookieDecoder
      • io.netty.handler.codec.http.cookie.ServerCookieDecoder

  • public final class ServerCookieDecoder
extends CookieDecoder
    一个RFC6265兼容cookie解码器将被用于服务器端。 只有名称和值字段是预期的，所以旧字段不填充（路径，域等）。 旧的RFC2965饼干仍然支持，旧的字段将被忽略。
