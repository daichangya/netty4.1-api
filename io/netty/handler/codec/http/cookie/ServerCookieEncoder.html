io.netty.handler.codec.http.cookie

Class ServerCookieEncoder

  • java.lang.Object
    • io.netty.handler.codec.http.cookie.CookieEncoder
      • io.netty.handler.codec.http.cookie.ServerCookieEncoder

  • public final class ServerCookieEncoder
extends CookieEncoder
    一个RFC6265兼容的Cookie编码器将被用于服务器端，所以一些字段被发送（版本通常被忽略）。 由于Netty Cookie将Expires和MaxAge合并为一个字段，因此只发送Max-Age字段。 请注意，多个Cookie必须作为单独的“Set-Cookie”标题发送。 
      // Example
 HttpResponse res = ...;
 res.setHeader("Set-Cookie", ServerCookieEncoder.encode("JSESSIONID", "1234"));
    另请参见：
    ServerCookieDecoder

    • 字段详细信息

      • STRICT

        public static final ServerCookieEncoder STRICT
        验证名称和值字符的严格编码器位于RFC6265中定义的有效范围内，并且（对于接受多个cookie的方法）只有一个cookie以任何给定名称进行编码。 （如果多个cookie具有相同的名称，则最后一个是编码的。）

      • LAX

        public static final ServerCookieEncoder LAX
        Lax实例不验证名称和值，并允许具有相同名称的多个Cookie。
