      • setMaxAge

        void setMaxAge(long maxAge)
        以秒为单位设置此Cookie的最大年龄。 如果指定的年龄为0 ，则该Cookie将被浏览器自动删除，因为它将立即过期。 如果UNDEFINED_MAX_AGE指定，这Cookie当浏览器关闭时将被删除。
        参数
        maxAge - 这个Cookie的最大年龄，以秒为单位

      • isHttpOnly

        boolean isHttpOnly()
        检查此Cookie是否只能通过HTTP访问。 如果返回true，则不能通过客户端脚本访问Cookie - 但仅限于浏览器支持。 欲了解更多信息，请查看here
        结果
        如果此Cookie仅为 HTTP，则为true，否则为false

      • setHttpOnly

        void setHttpOnly(boolean httpOnly)
        确定此Cookie是否仅HTTP。 如果设置为true，则不能通过客户端脚本访问此Cookie 。 但是，只有浏览器支持时才有效。 有关信息，请参阅here 。
        参数
        httpOnly - 如果Cookie仅为 HTTP， 则为 true，否则为false。
