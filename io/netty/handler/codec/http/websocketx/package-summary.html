Package io.netty.handler.codec.http.websocketx Description

Web Socket数据帧的编码器，解码器，握手及其相关消息类型。

该软件包支持不同的网络套接字规范版本（因此是X后缀）。 目前支持的规格是：

  • draft-ietf-hybi-thewebsocketprotocol-00
  • draft-ietf-hybi-thewebsocketprotocol-07
  • draft-ietf-hybi-thewebsocketprotocol-10
  • RFC 6455 （原来是draft-ietf-hybi-thewebsocketprotocol-17 ）

有关向您的HTTP服务器添加Web Socket支持的详细说明，请查看位于io.netty.example.http.websocket包中的WebSocketServerX示例。
