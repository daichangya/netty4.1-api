      • WebSocketServerHandshaker

        protected WebSocketServerHandshaker(WebSocketVersion version,
                                    java.lang.String uri,
                                    java.lang.String subprotocols,
                                    int maxFramePayloadLength)
        指定目标Web套接字位置的构造函数
        参数
        version - 协议版本
        uri - 网络套接字通信的URL。 例如“ws：//myhost.com/mypath”。 后续的网络套接字帧将被发送到这个URL。
        subprotocols - 支持的协议的CSV。 如果不支持子协议，则为空。
        maxFramePayloadLength - 帧有效负载的最大长度
