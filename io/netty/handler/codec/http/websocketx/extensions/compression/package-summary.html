Package io.netty.handler.codec.http.websocketx.extensions.compression Description

编码器，解码器，握手处理大多数常见的WebSocket压缩扩展。

该软件包支持不同的Web套接字扩展。 目前支持的规格是：

  • permessage-deflate
  • perframe-deflate and x-webkit-deflate-frame

请参阅io.netty.example.http.websocketx.client.WebSocketClient和io.netty.example.http.websocketx.html5.WebSocketServer的使用说明。
