Package io.netty.handler.codec

  • 异常摘要  
    异常 描述
    CodecException
    一个由编解码器引发的 异常 。
    CorruptedFrameException
    一个DecoderException当收到的帧数据不能被入站处理程序解码时抛出。
    DecoderException
    由解码器引发的CodecException 。
    EncoderException
    由编码器抛出的CodecException 。
    MessageAggregationException
    由于意外的消息顺序导致聚合失败时引发MessageAggregator 。
    PrematureChannelClosureException
    CodecException Channel在编解码器完成处理当前消息（例如在等待请求时缺少响应）之前意外关闭时引发的。
    TooLongFrameException
    一个DecoderException当解码帧的长度大于允许的最大值时抛出。
    UnsupportedMessageTypeException
    如果编解码器接收到不支持的消息则抛出。

Package io.netty.handler.codec Description

可扩展解码器及其常见实现，用于处理基于流的传输（如TCP / IP）中发现的数据包碎片和重新组装问题。
