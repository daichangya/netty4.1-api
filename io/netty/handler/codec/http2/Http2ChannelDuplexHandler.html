Class Http2ChannelDuplexHandler

  • java.lang.Object
    • io.netty.channel.ChannelHandlerAdapter

  • All Implemented Interfaces:
    ChannelHandler ， ChannelInboundHandler ， ChannelOutboundHandler 

    @UnstableApi
public abstract class Http2ChannelDuplexHandler
extends ChannelDuplexHandler
    A ChannelDuplexHandler为HTTP / 2提供附加功能。 具体来说，它允许：
    • 使用newStream()创建新的出站流。
    • 使用forEachActiveStream(Http2FrameStreamVisitor)迭代所有活动流。

    在添加该处理程序之前， Http2FrameCodec需要成为ChannelPipeline的一部分，否则将抛出IllegalStateException 。
