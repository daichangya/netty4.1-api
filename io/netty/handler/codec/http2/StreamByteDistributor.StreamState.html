io.netty.handler.codec.http2

Interface StreamByteDistributor.StreamState

  • Enclosing interface:
    StreamByteDistributor 

    public static interface StreamByteDistributor.StreamState
    该流的状态信息，指示当前可流式传输的字节数。 这提供给StreamByteDistributor.updateStreamableBytes(StreamState)方法。

      • stream

        Http2Stream stream()
        获取这个状态关联的流。

      • pendingBytes

        long pendingBytes()
        获取此流有待发送的字节量。 实际写入的金额不得超过windowSize() ！
        结果
        此流有待发送的字节数。
        另请参见：
        Http2CodecUtil#streamableBytes(StreamState)

      • hasFrame

        boolean hasFrame()
        指示是否存在等待此流的帧。

      • windowSize

        int windowSize()
        流的流量控制窗口的大小（以字节为单位）。 写入的金额不得超过此金额！

        StreamByteDistributor需要知道流的窗口大小，以避免在窗口大小为负时分配字节。 窗口大小为0对于确定一个流是否有机会编写一个空帧也是很重要的，并且还可以实现优化，比如在某些情况下不写空帧（不写头，直到数据也可以被写入）。

        结果
        流的流量控制窗口的大小。
        另请参见：
        Http2CodecUtil#streamableBytes(StreamState)
