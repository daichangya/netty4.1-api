Interface Http2Connection

  • 所有已知实现类：

    @UnstableApi
public interface Http2Connection
    管理器用于与远程端点建立HTTP / 2连接的状态。

    • 方法详细信息

      • close

        Future<java.lang.Void> close(Promise<java.lang.Void> promise)
        关闭此连接。 在此点之后不能创建更多的新流，并且所有存在的流（活动或其他）都将被关闭并删除。

        请注意，如果通过forEachActiveStream(Http2StreamVisitor)迭代活动流并抛出异常，则需要再次调用此方法以确保关闭完成。

        参数
        promise - 将在所有流都被删除并且收到通知时完成。
        结果
        未来将在所有流都被删除并且听众已被通知时完成。

      • newKey

        void removeListener(Http2Connection.Listener listener)
