    • 方法详细信息

      • channelHandlerContext

        ChannelHandlerContext channelHandlerContext()
        获取应用流量控制的ChannelHandlerContext 。

        这仅适用于Http2RemoteFlowController.FlowControlled实现。 谨慎使用。

        结果
        要应用流量控制的ChannelHandlerContext 。

      • addFlowControlled

        void addFlowControlled(Http2Stream stream,
                       Http2RemoteFlowController.FlowControlled payload)
        排队有效载荷以传输到远程端点。 不能保证数据何时被写入或将如何分配给帧。 发送之前。

        除非调用writePendingBytes()，否则写入不会实际发生。

        参数
        stream - 主题流。 一定不能是连接流对象。
        payload - 按流量控制会计和订购规则编写的有效负载。

      • hasFlowControlled

        boolean hasFlowControlled(Http2Stream stream)
        确定stream是否有任何Http2RemoteFlowController.FlowControlled当前排队的帧。
        参数
        stream - 检查流是否有流量控制帧的流。
        结果
        true如果stream有任何Http2RemoteFlowController.FlowControlled帧当前排队。

      • writePendingBytes

        void writePendingBytes()
                throws Http2Exception
        将流量控制器中待处理的所有数据写入流量控制限制。
        异常
        Http2Exception - 如果发生协议相关错误则抛出。

      • listener

        void listener(Http2RemoteFlowController.Listener listener)
        在流量控制器上设置活动监听器。
        参数
        listener - 通知何时写入发生，可以是 null 。

      • isWritable

        boolean isWritable(Http2Stream stream)
        确定stream是否剩余字节用于流量控制窗口。

        请注意，此方法尊重通道可写性。 该通道必须可写入此方法才能返回true 。

        参数
        stream - 要测试的流。
        结果
        true如果 stream剩余的字节用于流控制窗口并且通道可写，则 false否则为 false 。

      • channelWritabilityChanged

        void channelWritabilityChanged()
                        throws Http2Exception
        通知 channelHandlerContext()的可写性已更改。
        异常
        Http2Exception - 如果由于此调用而发生任何写入并遇到错误。

      • updateDependencyTree

        void updateDependencyTree(int childStreamId,
                          int parentStreamId,
                          short weight,
                          boolean exclusive)
        显式更新依赖关系树。 此方法独立于流状态更改而被调用。
        参数
        childStreamId - 与子流相关联的流标识符。
        parentStreamId - 与父流关联的流标识符。 可能是0 ，使childStreamId和直接子连接。
        weight - 用于parentStreamId相对于其他子流的parentStreamId 。 该值必须介于1和256（含）之间。
        exclusive - 如果 childStreamId应该是 childStreamId的唯一依赖 parentStreamId 。
