io.netty.handler.codec.http2

Interface StreamByteDistributor

  • 所有已知实现类：
    UniformStreamByteDistributor ， WeightedFairQueueByteDistributor 

    @UnstableApi
public interface StreamByteDistributor
    一个对象（由远程流控制使用），负责分配要写入连接流中的字节。

      • updateStreamableBytes

        void updateStreamableBytes(StreamByteDistributor.StreamState state)
        当流的流传输字节已更改时调用。 在第一次为给定流调用此方法之前，假定流没有可流式字节。

      • updateDependencyTree

        void updateDependencyTree(int childStreamId,
                          int parentStreamId,
                          short weight,
                          boolean exclusive)
        显式更新依赖关系树。 此方法独立于流状态更改而被调用。
        参数
        childStreamId - 与子流相关联的流标识符。
        parentStreamId - 与父流关联的流标识符。 可能是0 ，使childStreamId和直接子连接。
        weight - 用于parentStreamId相对于其他子流的parentStreamId 。 该值必须介于1和256（含）之间。
        exclusive - 如果 childStreamId应该是 childStreamId的独占依赖 parentStreamId 。

      • distribute

        boolean distribute(int maxBytes,
                   StreamByteDistributor.Writer writer)
            throws Http2Exception
        将最多maxBytes给那些包含流传输字节的流并遍历这些流以写入适当的字节。 遍历流的标准是未定义的，并且由实现决定何时停止在给定的流。

        流式字节不会通过调用此方法自动更新。 调用者通过调用updateStreamableBytes(StreamState)来指示写入之后可流式传输的字节数。

        参数
        maxBytes - 要写入的最大字节数。
        结果
        true是否还有尚未写入的流式字节，否则为 false 。
        异常
        Http2Exception - 如果发生内部异常并且内部连接状态将被破坏。
