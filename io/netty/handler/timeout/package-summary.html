Package io.netty.handler.timeout

使用Timer添加对读写超时和空闲连接通知的支持 。

See: 描述

  • 类摘要  
    Class 描述
    IdleStateEvent
    Channel空闲时由IdleStateHandler触发的用户事件。
    IdleStateHandler
    当Channel尚未执行读取，写入操作或同时执行两项操作时触发IdleStateEvent 。
    ReadTimeoutHandler
    在特定时间段内未读取数据时引发ReadTimeoutException 。
    WriteTimeoutHandler
    当写操作在某段时间内无法完成时引发WriteTimeoutException 。
  • 枚举摘要  
    Enum 描述
    IdleState
    代表Enum的空闲状态的Enum 。
  • 异常摘要  
    异常 描述
    ReadTimeoutException
    甲TimeoutException由凸起ReadTimeoutHandler时没有数据的一定时间内读出。
    TimeoutException
    A TimeoutException在某段时间内没有数据被读取或写入时。
    WriteTimeoutException
    当在特定时间段内没有写入数据时， TimeoutException由WriteTimeoutHandler产生。
