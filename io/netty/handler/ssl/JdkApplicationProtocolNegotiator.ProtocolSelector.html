io.netty.handler.ssl

Interface JdkApplicationProtocolNegotiator.ProtocolSelector

  • Enclosing interface:
    JdkApplicationProtocolNegotiator 

    public static interface JdkApplicationProtocolNegotiator.ProtocolSelector
    定义SSL握手过程中应用程序协议选择器角色的接口。 每个SSL握手都会调用unsupported()或select(List) 。

    • 方法摘要

      所有方法  接口方法  抽象方法 
      Modifier and Type Method and Description
      java.lang.String select(java.util.List<java.lang.String> protocols)
      调用Callback从 protocols选择应用程序级协议。
      void unsupported()
      调用回调让应用程序知道对方不支持此ApplicationProtocolNegotiator 。

    • 方法详细信息

      • unsupported

        void unsupported()
        调用回调函数，让应用程序知道对方不支持这个ApplicationProtocolNegotiator 。

      • select

        java.lang.String select(java.util.List<java.lang.String> protocols)
                 throws java.lang.Exception
        调用Callback从 protocols提供的应用程序级协议中选择。
        参数
        protocols - 由协议广告商发送的协议
        结果
        由此JdkApplicationProtocolNegotiator.ProtocolSelector选择的协议。 值为null表示没有选择协议，但握手不应失败。 握手失败的决定留给另一端来协商SSL握手。
        异常
        java.lang.Exception - 如果 protocols提供授权失败的SSL握手致命警报。
