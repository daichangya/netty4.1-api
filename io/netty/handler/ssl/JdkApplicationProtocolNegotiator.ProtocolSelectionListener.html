io.netty.handler.ssl

Interface JdkApplicationProtocolNegotiator.ProtocolSelectionListener

  • Enclosing interface:
    JdkApplicationProtocolNegotiator 

    public static interface JdkApplicationProtocolNegotiator.ProtocolSelectionListener
    一个监听者被其同伴选择的协议通知。 每个SSL握手都会调用unsupported()或selected(String)方法。

    • 方法摘要

      所有方法  接口方法  抽象方法 
      Modifier and Type Method and Description
      void selected(java.lang.String protocol)
      调用回调让该应用程序知道对等方选择的协议。
      void unsupported()
      调用回调函数让应用程序知道对方不支持这个ApplicationProtocolNegotiator 。

    • 方法详细信息

      • unsupported

        void unsupported()
        调用回调让应用程序知道对方不支持此ApplicationProtocolNegotiator 。

      • selected

        void selected(java.lang.String protocol)
       throws java.lang.Exception
        调用回调让该应用程序知道对等方选择的协议。
        参数
        protocol - 由对等方选择的协议。 可能是null或者应用协商协议支持为空。
        异常
        java.lang.Exception - 如果选择的协议不可接受并且期望的行为是通过致命警报失败握手，则可能会引发此问题。
