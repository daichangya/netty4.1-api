Package io.netty.resolver

将表示端点名称的任意字符串解析为地址。

See: 描述

  • 接口摘要  
    接口 描述
    AddressResolver<T extends java.net.SocketAddress>
    解决未解决的可能性 SocketAddress 。
    HostsFileEntriesResolver
    根据主机文件条目解析主机名。
    NameResolver<T>
    将表示端点名称的任意字符串解析为地址。
  • 类摘要  
    Class 描述
    AbstractAddressResolver<T extends java.net.SocketAddress>
    一个骨架AddressResolver的实现。
    AddressResolverGroup<T extends java.net.SocketAddress>
    创建并管理NameResolver，以便每个EventExecutor都有自己的解析器实例。
    CompositeNameResolver<T>
    一个组合SimpleNameResolver ，它根据NameResolver的序列解析主机名。
    DefaultAddressResolverGroup
    AddressResolverGroup DefaultNameResolver s。
    DefaultHostsFileEntriesResolver
    默认HostsFileEntriesResolver仅解析主机文件条目一次。
    DefaultNameResolver
    一个使用JDK的内置域名查找机制解决的InetNameResolver 。
    HostsFileEntries
    主机文件条目的容器
    HostsFileParser
    主机文件的解析器。
    InetNameResolver
    解决InetAddress骨架NameResolver实现。
    InetSocketAddressResolver
    A AbstractAddressResolver ，解决InetSocketAddress 。
    NoopAddressResolver
    A AddressResolver不执行任何决议，但始终报告成功的解决方案。
    NoopAddressResolverGroup
    AddressResolverGroup NoopAddressResolver s。
    RoundRobinInetAddressResolver
    A NameResolver解析InetAddress并通过在SimpleNameResolver.resolve(String)和SimpleNameResolver.resolve(String, Promise)中随机选择单个地址（如果NameResolver返回多个地址）来强制循环播放 。
    SimpleNameResolver<T>
    一个骨架NameResolver的实现。
