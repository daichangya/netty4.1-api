Class DefaultDnsCache

  • java.lang.Object
    • io.netty.resolver.dns.DefaultDnsCache

    @UnstableApi
public class DefaultDnsCache
extends java.lang.Object
implements DnsCache
    默认实现DnsCache ，由ConcurrentMap支持。 如果使用其他DnsRecord ，则不会发生缓存。
