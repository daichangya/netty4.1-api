io.netty.resolver.dns

Interface DnsServerAddressStreamProvider

  • 所有已知实现类：
    DefaultDnsServerAddressStreamProvider ， MultiDnsServerAddressStreamProvider ， SequentialDnsServerAddressStreamProvider ， SingletonDnsServerAddressStreamProvider ， UnixResolverDnsServerAddressStreamProvider 

    @UnstableApi
public interface DnsServerAddressStreamProvider
    提供一个机会来覆盖哪个DnsServerAddressStream用于解析特定的主机名。

    例如，这可以用来表示/etc/resolv.conf和/etc/resolver 。

    • 方法摘要

      所有方法  接口方法  抽象方法 
      Modifier and Type Method and Description
      DnsServerAddressStream nameServerAddressStream(java.lang.String hostname)
      询问该提供商的名称服务器以查询 hostname 。

    • 方法详细信息

      • nameServerAddressStream

        DnsServerAddressStream nameServerAddressStream(java.lang.String hostname)
        询问该提供商的名称服务器以查询 hostname 。
        参数
        hostname - 要查找要使用的DNS服务器的主机名。 如果这是要查询的最终DnsServerAddressStreamProvider ，则一般为空字符串或'.'对应于默认DnsServerAddressStream 。
        结果
        DnsServerAddressStream应该用来解决hostname 。
