Package io.netty.resolver.dns

Java的内置域名查找机制的一种替代方法，可以异步解析域名，同样支持任意DNS记录类型的查询。

See: 描述

  • 接口摘要  
    接口 描述
    DnsCache
    DNS解析条目的缓存。
    DnsCacheEntry
    表示可缓存的先前DNS查询的结果。
    DnsQueryLifecycleObserver
    该界面提供对各个DNS查询的可视性。
    DnsQueryLifecycleObserverFactory
    用于生成DnsQueryLifecycleObserver的新实例。
    DnsServerAddressStream
    无限的DNS服务器地址流。
    DnsServerAddressStreamProvider
    提供一个机会来覆盖哪个DnsServerAddressStream用于解析特定的主机名。
  • 类摘要  
    Class 描述
    BiDnsQueryLifecycleObserver
    将两个DnsQueryLifecycleObserver组合成一个DnsQueryLifecycleObserver 。
    BiDnsQueryLifecycleObserverFactory
    将两个DnsQueryLifecycleObserverFactory组合成一个DnsQueryLifecycleObserverFactory 。
    DefaultDnsCache
    默认实现DnsCache ，由ConcurrentMap支持。
    DefaultDnsServerAddressStreamProvider
    DnsServerAddressStreamProvider将使用预定义的默认DNS服务器用于DNS解析。
    DnsAddressResolverGroup
    A AddressResolverGroup DnsNameResolver s。
    DnsNameResolver
    基于DNS的InetNameResolver 。
    DnsNameResolverBuilder
    DnsNameResolver建造者。
    DnsServerAddresses
    提供无限序列的DNS服务器地址到DnsNameResolver 。
    DnsServerAddressStreamProviders
    与DnsServerAddressStreamProvider相关的实用方法。
    MultiDnsServerAddressStreamProvider
    甲DnsServerAddressStreamProvider其中通过的集合迭代DnsServerAddressStreamProvider直到第一非null结果被发现。
    NoopDnsCache
    实际上从不缓存任何东西的noop DNS缓存。
    NoopDnsQueryLifecycleObserverFactory  
    RoundRobinDnsAddressResolverGroup
    一个AddressResolverGroup的DnsNameResolver s，支持随机选择目标地址（如果有多个由名称服务器提供）。
    SequentialDnsServerAddressStreamProvider
    一个DnsServerAddressStreamProvider ，它由一个连续的DNS服务器列表支持。
    SingletonDnsServerAddressStreamProvider
    DnsServerAddressStreamProvider总是使用单个DNS服务器进行解析。
    UnixResolverDnsServerAddressStreamProvider
    能够解析文件（如 /etc/resolv.conf和 /etc/resolver）以尊重系统默认域服务器。
  • 异常摘要  
    异常 描述
    DnsNameResolverException
    一个RuntimeException时提出DnsNameResolver未能履行一项成功的查询。
    DnsNameResolverTimeoutException
    一个DnsNameResolverException时提出DnsNameResolver失败，因为超时的执行成功的查询。

Package io.netty.resolver.dns Description

An alternative to Java's built-in domain name lookup mechanism that resolves a domain name asynchronously, which supports the queries of an arbitrary DNS record type as well.
