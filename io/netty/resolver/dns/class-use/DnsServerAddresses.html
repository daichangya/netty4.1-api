      Modifier and Type Method and Description
      static DnsServerAddresses DnsServerAddresses.defaultAddresses()
      已过时。
      使用DefaultDnsServerAddressStreamProvider.defaultAddresses() 。

      返回依次产生系统DNS服务器地址的DnsServerAddresses 。 如果它无法从环境中检索系统DNS服务器地址列表，它将使用Google公有DNS服务器的地址"8.8.8.8"和"8.8.4.4" 。

      此方法与以下代码具有相同的效果： 

        DnsServerAddresses.sequential(DnsServerAddresses.defaultAddressList());
