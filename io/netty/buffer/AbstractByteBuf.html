        public ByteBuf setIndex(int readerIndex,
                        int writerIndex)
        从类复制的描述： ByteBuf
        readerIndex设置此缓冲区的readerIndex和writerIndex 。 当您不得不担心ByteBuf.readerIndex(int)和ByteBuf.writerIndex(int)方法的调用顺序时，此方法非常有用。 例如，以下代码将失败：

 // 0, 0 and 8 respectively.

 // readerIndex (2) cannot be greater than the current writerIndex (0).

 buf.readLong();

 // writerIndex (4) cannot be less than the current readerIndex (8).
 buf.writerIndex(4);
 buf.readerIndex(2);
