Class Signal

    Constant < Signal >，java.io.Serializable，java.lang.Comparable < Signal > 

    public final class Signal
extends java.lang.Error
implements Constant<Signal>
    一个特殊的Error ，用于通过投掷来发信号通知某个状态或请求。 Signal有一个空的堆栈跟踪，并没有理由保存实例化开销。
