io.netty.util

Interface Timeout


  • public interface Timeout
    与Timer返回的TimerTask关联的句柄 。

    • 方法摘要

      所有方法  接口方法  抽象方法 
      Modifier and Type Method and Description
      boolean cancel()
      尝试取消与此句柄关联的TimerTask 。
      boolean isCancelled()
      当且仅当与此句柄关联的TimerTask已被取消时才返回true 。
      boolean isExpired()
      当且仅当与此句柄关联的TimerTask已过期时才返回true 。
      TimerTask task()
      返回与此句柄关联的TimerTask 。
      Timer timer()
      返回创建此句柄的Timer 。

    • 方法详细信息

      • timer

        Timer timer()
        返回创建此句柄的Timer 。

      • task

        TimerTask task()
        返回与此句柄关联的TimerTask 。

      • isExpired

        boolean isExpired()
        当且仅当与此句柄关联的TimerTask已过期时才返回true 。

      • isCancelled

        boolean isCancelled()
        当且仅当与此句柄关联的TimerTask已被取消时返回true 。

      • cancel

        boolean cancel()
        试图取消与此句柄关联的TimerTask 。 如果任务已经执行或取消，它将返回而没有副作用。
        结果
        如果取消成功完成，则为true，否则为false
