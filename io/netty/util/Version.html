Class Version

  • java.lang.Object
    • io.netty.util.Version

  • public final class Version
extends java.lang.Object
    检索可用Netty工件的版本信息。

    该类从构建时生成的META-INF/io.netty.versions.properties检索版本信息。 请注意，根据您的环境（如指定的ClassLoader ，当前的SecurityManager ，可能无法完全检索信息。
