        public static boolean isIpV4StackPreferred()

      • getByName

        public static java.net.Inet6Address getByName(java.lang.CharSequence ip)
        返回CharSequence IP地址的Inet6Address表示形式。

        此方法将所有IPv4类型的地址视为“IPv4映射”（请参阅getByName(CharSequence, boolean) ）

        参数
        ip - CharSequence IP地址被转换为一个 Inet6Address
        结果
        Inet6Address如果不是有效的IP地址，则表示 ip或 null 。

      • getByName

        public static java.net.Inet6Address getByName(java.lang.CharSequence ip,
                                              boolean ipv4Mapped)
        返回CharSequence IP地址的Inet6Address表示形式。

        ipv4Mapped参数指定应如何处理IPv4地址。 支持rfc 4291 section 2中定义的“IPv4映射”格式。

        参数
        ip - CharSequence IP地址被转换为一个 Inet6Address
        ipv4Mapped -
        • true允许将IPv4映射输入转换为Inet6Address
        • false IPv4映射地址视为无效。
        结果
        Inet6Address如果不是有效的IP地址，则表示 ip或 null 。

      • toAddressString

        public static java.lang.String toAddressString(java.net.InetAddress ip)
        返回String表示InetAddress 。
        • Inet4Address结果与InetAddress.getHostAddress()相同
        • Inet6Address结果符合rfc 5952 section 4

        输出不包含范围ID。

        参数
        ip - InetAddress将被转换为地址字符串
        结果
        String包含文本格式的IP地址

      • toAddressString

        public static java.lang.String toAddressString(java.net.InetAddress ip,
                                               boolean ipv4Mapped)
        返回String表示InetAddress 。
        • Inet4Address结果与InetAddress.getHostAddress()完全相同
        • 如果ipv4Mapped为假，则Inet6Address结果遵循rfc 5952 section 4 。 如果ipv4Mapped为真，则将支持来自rfc 4291 section 2的 “IPv4映射”格式。 压缩结果将始终服从rfc 5952 section 4中定义的压缩规则

        输出不包含范围ID。

        参数
        ip - InetAddress将被转换为地址字符串
        ipv4Mapped -
        • true从严格的RFC 5952流浪和支持中定义的“IPv4映射”格式rfc 4291 section 2 ，同时仍然在遵循最新准则rfc 5952 section 4
        • false严格遵循rfc 5952
        结果
        String包含文本格式的IP地址
