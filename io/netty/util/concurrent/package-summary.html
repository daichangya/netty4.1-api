Package io.netty.util.concurrent

并发/异步任务的实用程序类。

See: 描述

  • 接口摘要  
    接口 描述
    EventExecutor
    EventExecutor是一个特殊的EventExecutorGroup ，它提供了一些方便的方法来查看Thread是否在事件循环中执行。
    EventExecutorChooserFactory
    创建新的EventExecutorChooserFactory.EventExecutorChooser s的工厂。
    EventExecutorChooserFactory.EventExecutorChooser
    选择下一个EventExecutor使用。
    EventExecutorGroup
    EventExecutorGroup负责通过其EventExecutorGroup.next()方法提供EventExecutor的使用。
    Future<V>
    异步操作的结果。
    FutureListener<V>
    为便于隐藏类型参数的子类型GenericFutureListener 。
    GenericFutureListener<F extends Future<?>>
    听取Future的结果。
    GenericProgressiveFutureListener<F extends ProgressiveFuture<?>>  
    OrderedEventExecutor
    用于EventExecutor的标记接口，将以有序/串行方式处理所有提交的任务。
    ProgressiveFuture<V>
    A Future用于指示操作的进度。
    ProgressivePromise<V>
    特别ProgressiveFuture这是可写的。
    Promise<V>
    特别Future这是可写的。
    RejectedExecutionHandler
    类似于RejectedExecutionHandler但是特定于SingleThreadEventExecutor 。
    ScheduledFuture<V>
    预定的异步操作的结果。
    ThreadProperties
    公开 Thread详细信息。
  • 类摘要  
    Class 描述
    AbstractEventExecutor
    EventExecutor实现的抽象基类。
    AbstractEventExecutorGroup
    EventExecutorGroup实现的抽象基类。
    AbstractFuture<V>
    摘要Future不允许取消的实现。
    AbstractScheduledEventExecutor
    希望支持调度的抽象基类EventExecutor 。
    CompleteFuture<V>
    代表已完成的Future的骨架Future实现。
    DefaultEventExecutor
    默认SingleThreadEventExecutor实现， 它只是以串行方式执行所有提交的任务。
    DefaultEventExecutorChooserFactory
    使用简单的循环法选择下一个EventExecutor的默认实现。
    DefaultEventExecutorGroup
    默认实现MultithreadEventExecutorGroup ，它将使用DefaultEventExecutor实例来处理任务。
    DefaultProgressivePromise<V>  
    DefaultPromise<V>  
    DefaultThreadFactory
    具有简单命名规则的 ThreadFactory实现。
    FailedFuture<V>
    已经失败的CompleteFuture 。
    FastThreadLocal<V>
    的一个特殊变种ThreadLocal从访问时产生更高的访问性能FastThreadLocalThread 。
    FastThreadLocalThread
    一个特殊的Thread ，可以快速访问FastThreadLocal变量。
    GlobalEventExecutor
    单线程单身人士EventExecutor 。
    ImmediateEventExecutor
    在调用者的线程中执行 Runnable对象。
    ImmediateExecutor
    Executor ，它们在调用者线程中执行任务。
    MultithreadEventExecutorGroup
    EventExecutorGroup实现的抽象基类， 它们同时处理具有多个线程的任务。
    NonStickyEventExecutorGroup
    EventExecutorGroup将保留Runnable执行顺序，但不保证EventExecutor （因此Thread ）将用于执行Runnable s。
    PromiseAggregator<V,F extends Future<V>> Deprecated
    使用PromiseCombiner GenericFutureListener实现，通过监听单个Future s并在所有Future完成时产生聚合结果（成功/失败），将多个Future合并为一个。
    PromiseCombiner
    一个承诺组合器监视一些离散期货的结果，然后在所有的合并期货结束时通知最终的总承诺。
    PromiseNotifier<V,F extends Future<V>>
    GenericFutureListener执行其他Promise s并完成后通知他们。
    RejectedExecutionHandlers
    公开创建不同RejectedExecutionHandler的辅助方法。
    SingleThreadEventExecutor
    OrderedEventExecutor的抽象基类，它在一个线程中执行所有提交的任务。
    SucceededFuture<V>
    已经成功的CompleteFuture 。
    ThreadPerTaskExecutor  
    UnaryPromiseNotifier<T>  
    UnorderedThreadPoolEventExecutor
    EventExecutor实现，因为可能有多个线程正在执行这些任务，所以对提交的任务执行的排序没有任何保证。
  • 异常摘要  
    异常 描述
    BlockingOperationException
    当用户在事件循环线程中时，用户执行阻止操作时引发的 IllegalStateException 。
