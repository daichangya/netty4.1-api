io.netty.bootstrap

Class Bootstrap

  • java.lang.Object
    • io.netty.bootstrap.AbstractBootstrap<Bootstrap,Channel>
      • io.netty.bootstrap.Bootstrap
  • All Implemented Interfaces:
    java.lang.Cloneable 

    public class Bootstrap
extends AbstractBootstrap<Bootstrap,Channel>
    一个Bootstrap ，可以很容易地引导Channel用于客户端。

    AbstractBootstrap.bind()方法可与无连接传输（如数据报（UDP））结合使用。 对于常规的TCP连接，请使用提供的connect()方法。

      • clone

        public Bootstrap clone()
        从类复制的描述： AbstractBootstrap
        返回具有相同配置的此引导程序的深层克隆。 使用类似设置制作多个Channel时，此方法很有用。 请注意，这种方法不能深入而浅薄地克隆EventLoopGroup ，从而使该组成为共享资源。
        Specified by:
        clone的 AbstractBootstrap<Bootstrap,Channel>
